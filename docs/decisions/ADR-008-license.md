# ADR-008: MIT License, Public Repository

**Status:** Accepted

## Context
The repository access policy (public vs private) is unspecified. We need a license that
allows TP to use, review, and build on the code without legal friction.

## Decision
- **License:** MIT — permissive, allows reuse, modification, and distribution with attribution
- **Visibility:** Public — reviewers can clone without requiring a GitHub invitation

## Consequences
- ✅ No legal blocking for TP to use or extend the code
- ✅ Industry standard for technical prototypes and shared review
- ✅ Public repo eliminates access friction during review
- ❌ Code is publicly visible — acceptable; this is a technical demonstration, not proprietary IP
